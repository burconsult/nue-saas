Introduction

The Nue SAAS Boilerplate is a complete, production-ready foundation for building SAAS applications. Built with Nue.js, it provides all the essential features you need to launch your SAAS product quickly.

Key Features

  • Complete authentication system with OAuth support
  • Stripe integration for billing and subscriptions
  • Modern, responsive dashboard
  • Admin panel for user management
  • API with rate limiting and authentication
  • Real-time analytics and monitoring

Quick Start

Get up and running in minutes with these simple steps:

# Clone the repository
git clone https://github.com/nuejs/nue-saas-boilerplate.git
cd nue-saas-boilerplate

# Install dependencies
npm install

# Start development server
npm run dev

Your application will be available at http://localhost:8080

Installation

Detailed installation instructions for different environments.

Prerequisites

  • Node.js 18+ or Bun 1.0+
  • PostgreSQL or MySQL database
  • Stripe account for payments
  • Email service (SendGrid, Mailgun, etc.)

Configuration

Configure your application with environment variables and settings.

Environment Variables

DATABASE_URL=postgresql://user:password@localhost:5432/database
STRIPE_SECRET_KEY=sk_test_...
STRIPE_PUBLISHABLE_KEY=pk_test_...
EMAIL_SERVICE_API_KEY=your-email-service-key
JWT_SECRET=your-secret-key
